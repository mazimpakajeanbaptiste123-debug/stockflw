Free real-time sync across all your devices — works offline too

1Create Free Firebase Project

Go to console.firebase.google.com → Add project → Name it stockflow-app → Disable Analytics → Create project

2Enable Firestore Database

In your project → Build → Firestore Database → Create database → Choose Start in test mode → Select your region → Enable

3Get Your Config

Go to Project Settings ⚙️ (gear icon) → General → scroll to Your apps → Click </> Web → Register app → Copy the firebaseConfig object values below:
